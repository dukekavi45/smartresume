# Smart Resume Analyzer & Company Recommendation System

## 📌 Project Overview
The Smart Resume Analyzer is an AI-powered web application that analyzes resumes and recommends suitable companies and job roles based on skill matching. It also identifies missing skills to help candidates improve their employability.

The system supports multiple domains including IT, Non-IT, Core Engineering, Healthcare, Education, and Business roles.

---

## 🚀 Features
- User Signup & Login Authentication
- Resume Upload (PDF / DOCX)
- Automatic Skill Extraction using NLP
- Company & Job Role Recommendation
- Match Percentage Calculation
- Skill Gap Identification
- Interactive Web Interface
- Multi-domain Job Dataset (200+ companies)
